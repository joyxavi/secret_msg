# Secret Message Sender

Secret Message Sender is a secure client-side web application that enables users to share confidential messages through encrypted, password-protected links. The application uses AES encryption to protect message content and supports additional security features such as message expiry and self-destruction after viewing.

This project was developed to demonstrate frontend security concepts, encryption techniques, and modern user interface design using core web technologies.





---

## Features

* AES-256 encryption for secure message protection
* Password-protected message access
* Self-destruct option after message is read
* Configurable message expiry timer
* Shareable encrypted link generation
* WhatsApp sharing integration
* Copy-to-clipboard functionality
* Responsive and modern user interface
* Fully client-side implementation (no backend required)

## How It Works

1. User enters a message and optional password
2. Message is encrypted using AES encryption
3. Encrypted data is embedded in a unique URL
4. Recipient opens the link and enters the password
5. Message is decrypted and displayed securely
6. Message expires or self-destructs if configured
